# Calm Rhythm 2 — Marketing URL website

Static HTML project for an App Store Marketing URL.

## Included pages

- `index.html` — landing page
- `support.html` — support page with form and email
- `privacy.html` — privacy summary with link to official policy
- `faq.html` — frequently asked questions
- `contact.html` — developer contact page
- `404.html` — fallback page

## Files

- `assets/css/styles.css`
- `assets/js/script.js`
- `assets/img/*`

## Customize

1. Upload the project to any static host.
2. Use the homepage URL as the Marketing URL in App Store Connect.
3. If you want, you can later add your App Store link as an extra button on `index.html`.
